pypegase.include package¶

This module contains a few codes that are compiled with cython to provide some significant speed up.

Note

When codes do not compile properly, codes fallback into pure python implementations

Configuration file could also force fallback to python

Module contents¶

C extensions to pyPegase In this model some compiled cython code will replace some python code when C interface is available.

In particular, the stellar library interpolator gains in speed when using a c-equivalent.
