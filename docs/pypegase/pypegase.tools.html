pypegase.tools.decorators module¶

This package groups a few simple decorators but especially useful during debugging phases

memoize():
(fast version) Caches a function’s return value each time it is called.
persistent_locals():
Keeps access to the local variables accessible after execution.
timeit():
Time a block of your code (also usable as a context manager, ‘with’).
run_async():
intended to make decorated functions to run in a separate thread (asynchronously).
trace():
Trace a function call and logs args, kwargs, result and optionaly execution time

Requires dependencies:

memtrace()
This decorator uses <memory_profiler> to trace line by line the memory

pypegase.tools.pbar module¶

Simple progressbar¶

This package implement a unique progress bar class that can be used to decorate an iterator, a function or even standalone.

The format of the meter is flexible and can display along with the progress meter, the running time, an eta, and the rate of the iterations.

An example is::
description [———-] k/n 10% [time: 00:00:00, eta: 00:00:00, 2.7 iters/sec]
class pypegase.tools.pbar.Pbar(maxval=None, desc=None, time=True, eta=True, rate=True, length=None, file=None, keep=True, mininterval=0.5, miniters=1, units='iters', **kwargs)[source]¶
